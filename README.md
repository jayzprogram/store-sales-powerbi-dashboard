# 🛒 Superstore Sales Dashboard

An interactive Power BI dashboard analyzing Superstore sales 2010–2014.

## 📊 Dashboard Preview
![Dashboard Preview](dashboard_preview.png)

## 📌 Features
- Sales & Profit by Region and Category
- Year-over-Year sales trends
- Customer segment analysis
- Quantity and shipping insights

## 🛠 Tools Used
- Power BI Desktop
- Dataset: Sample Superstore

## 📂 Files
| File | Description |
|------|-------------|
| `StoreSales_Dashboard.pbix` | Power BI dashboard file |
| `StoreSales_Report.pdf` | Exported PDF report |
| `dashboard_preview.png` | Dashboard screenshot |

## 🚀 How to Use
1. Download `StoreSales_Dashboard.pbix`
2. Open with Power BI Desktop
3. Explore the report!
